# Student-CRUD
Spring Boot CRUD application using:
- Spring Data JPA
- Native Queries
- REST APIs
- MySQL
- Lombok

Endpoints:
- GET /students
- GET /students/{rno}
- GET /students/technology/{tech}
- GET /students/filter?gender=&technology=
- POST /students
- POST /students/addAll
- PUT /students
- DELETE /students/{rno}
- DELETE /students/clear
